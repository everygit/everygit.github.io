# flex 滚动条

由于弹性布局，导致滚动条有点不一样

# 在全局的 flex 下，实现滚动条

当使用 overflow: auto 的时候，无法出现滚动条，而本身会发生变化，原因是由于父节点没有固定高度（主轴轴是横轴）导致，无法解释出本身盒子的大小。

解决方案是，通过给固定高度（主轴轴是横轴）或者父节点要增加 overflow: hidden 来约束高度。

# 在交叉轴无法出现滚动条

原因是 flex 的 align-items 默认值是 stretch 导致无法解释，将高度（交叉轴是纵向）或宽度（交叉轴是横向）设置为 0 即可。
