# IIS 配置 HTTPS

因为有些时候你无法选择技术栈，那么 IIS 的证书我们也需要关注

# 证书转换

crt 格式的证书并不能直接用于IIS，需要转换为pfx格式才可以

openssl pkcs12 -export -inkey server.key -in server.crt -out server.pfx

在转换的过程中会要求输入密码，这个密码请牢记，因为在证书导入的过程中需要使用

# 配置

打开IIS管理器

选择 “服务器证书”

点击导入

选择证书路径，输入密码，选择证书存储，选择 Web宿主 就可以了。

点击确定，可能会有警告，因为我们并未使用中间证书，直接用根证书签发的服务器证书，不过没关系，依然是可以用的。
