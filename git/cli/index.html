# 日志 log

# show

等效于

git log -p -1

某次提交的版本，某个文件的变化

git show commit-id filename

简化并格式化提交记录

git show -s --pretty=raw 2be7fcb476

# log

远程版本库的 log

git log origin/master

查看某个文件发生变动时的历史提交记录

git log filename

可以看到 fileName 相关的commit记录及修改内容

git log -p filename

根据作者查询

git log --author everygit

查看单行

git log --oneline

与上面的不太一样

git log --pretty=oneline

# graph

git log --online --graph --all
