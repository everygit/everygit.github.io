# 贮藏 stash

基于 stash 创建分支

git stash
git stash branch newb

将未跟踪的文件

git stash push -u -m "some desc"

不影响暂存区

git stash push -k

git stash drop stash@{1} # 删除指定储藏
git stash clear #  删除所有储藏
