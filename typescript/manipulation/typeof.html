# ReturnType<T>

type Predicate = (x: unknown) => boolean;
type K = ReturnType<Predicate>;

K 的类型是 boolean

注意 ReturnType 接受的是类型

function f() {
  return { x: 10, y: 3 };
}
type P = ReturnType<f>;

所以，这里不会得到预期的效果

function f() {
  return { x: 10, y: 3 };
}
type P = ReturnType<typeof f>;
// type p = {x:number, y: number}

要使用 typeof 才能获取 f 的类型
