# 安装 emqx

emqx 是一个集成了 mqtt 完整功能的一个平台工具

docker run -d --name emqx -p 18083:18083 -p 1883:1883 -p 8083:8083 emqx/emqx-ee:latest

访问地址：http://localhost:18083

账号：admin，密码：public
