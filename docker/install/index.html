# mac 安装

mac 相比 linux 来说有图形化管理界面，使用起来也比较友好，虽然命令也很简单

# 安装包

水果出了自家的芯片，所以现在安装包需要选择自己的芯片版本。老衲是价格敏感型人群，所以还是老旧的 intel 芯片。

# 安装

下载的文件为 dmg 文件，安装起来还是那么老套，就不说啥了

# 修改 docker 镜像

设置 -> Docker Engine

"registry-mirrors": [
    "https://xxxx.mirror.aliyuncs.com"
],

老衲用了阿里云的源，控制台中的 应用 -> 容器与中间件 -> 容器镜像服务 -> 镜像工具 -> 镜像加速器

然后复制连接就可以了。
