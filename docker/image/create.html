# 创建镜像

有很多中创建镜像的方式

# 从现有的容器

从已有的容器，可能又安装一些应用，比如 node 的镜像，安装很多的依赖，可以直接创建一个镜像

docker commit -m "sss" -a "sbt" 9216e4a2967a sbt0198/ubuntu:v3

9216e4a2967a 为容器 ID，而不是镜像 ID

# Dockerfile

先看命令：

docker build -t sbt0198/centos:6.7 .
  • -t 是指定要创建的镜像名
  • . 代表 Dockerfile 的路径
