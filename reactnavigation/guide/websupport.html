# Web 上的 React Navigation

注意:对 web 的支持是实验性的，工作正在进行中。它有缺陷，缺少许多功能，而且用于 web 集成的 API 在小版本中可能会改变。如果遇到 bug，请帮助我们测试并打开 bug 报告。

React Navigation 的 web 支持目前需要使用 React Native for web。这种方法让我们可以在 React Native 和 Web 上重用相同的代码。

目前，有以下功能:

  • 浏览器中的 URL 集成 (opens new window)
  • 访问链接 (opens new window)
  • 服务器渲染 (opens new window)

重要的是，使用链接作为导航的主要方式，而不是像 navigation.navigate 这样的导航操作。它将确保您的链接是正确的可用的网络。

一些导航器也配置不同的 web 或提供额外的 web 特定功能:

  1. 折叠项和底部选项卡导航器分别在折叠项侧边栏和选项卡栏中显示超链接。
  2. 在网页上使用时，抽屉和堆栈导航器上的滑动手势是不可用的。
  3. 默认情况下，堆栈导航器禁用页面转换动画，但它可以通过指定 animationEnabled: true 重新启用。

注意:不像 React Navigation 4，当你使用 React Native for web 时，你不需要安装一个单独的包来使用 web 集成。如果你安装了 @react-navigation/web 包，请卸载它，因为它不能与 React Navigation 5 一起使用。
