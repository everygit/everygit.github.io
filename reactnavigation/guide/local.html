# 本地化

英语只是世界上人们使用的众多语言之一(非常感谢Tower of Babel (opens new window))，有时候将我们的应用翻译成用户使用的语言是礼貌的，甚至是必要的。让我们来看看我们可以在 React Navigation 中做到这一点的一种方法——它不是唯一的方法，但它会做到这一点。类似于主题，我们将使用 React 的上下文 API，以便更容易地从各种组件访问翻译函数。

# 设置本地化库

我们需要使用某种库来存储翻译，并提供一个函数来访问它们，以及在没有定义特定语言时处理回退。本地化和国际化(i18n)通常是可以互换使用的，在下面的例子中，我们从 expo-localization 获取当前的 locale，并使用 i18n-js 库来管理翻译，没有什么特别的原因，只是因为它是可用的 —— 使用任何你喜欢的东西。

import * as Localization from "expo-localization"; // or whatever library you want
import i18n from "i18n-js"; // or whatever library you want

const en = {
  foo: "Foo",
  bar: "Bar {{someValue}}",
};

const fr = {
  foo: "Fou",
  bar: "Bár {{someValue}}",
};

i18n.fallbacks = true;
i18n.translations = { fr, en };

// This will log 'en' for me, as I'm an English speaker
console.log(Localization.locale);

# 将本地化库与导航连接起来

接下来，让我们将语言环境存储在根应用程序组件的状态中，然后通过 LocalizationContext 将其贯穿在整个应用程序中可用。

export const LocalizationContext = React.createContext();

export default function App() {
  const [locale, setLocale] = React.useState(Localization.locale);
  const localizationContext = React.useMemo(
    () => ({
      t: (scope, options) => i18n.t(scope, { locale, ...options }),
      locale,
      setLocale,
    }),
    [locale]
  );

  return (
    <LocalizationContext.Provider value={localizationContext}>
      <NavigationContainer>{/* Screen configuration */}</NavigationContainer>
    </LocalizationContext.Provider>
  );
}

现在在屏幕中，我们可以使用这些 LocalizationContext，如下所示:

function MyScreen() {
  const { t, locale, setLocale } = React.useContext(LocalizationContext);

  return (
    <View style={styles.container}>
      <Text style={styles.text}>
        Current locale: {locale}.{" "}
        {locale !== "en" && locale !== "fr"
          ? 'Translations will fall back to "en" because none available'
          : null}
      </Text>
      <Text>{t("bar", { someValue: Date.now() })}</Text>
      {locale === "en" ? (
        <Button title="Switch to French" onPress={() => setLocale("fr")} />
      ) : (
        <Button title="Switch to English" onPress={() => setLocale("en")} />
      )}
    </View>
  );
}

我们也可以在屏幕选项中使用它:

function MyStack() {
  const { t } = React.useContext(LocalizationContext);

  return (
    <Stack.Navigator>
      <Stack.Screen
        name="Home"
        component={MyScreen}
        options={{ title: t("foo") }}
      />
    </Stack.Navigator>
  );
}
