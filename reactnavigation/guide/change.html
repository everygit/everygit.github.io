# 当焦点屏幕改变时调用一个函数

在本指南中，在屏幕焦点时我们将调用一个函数或渲染某些内容。 当用户在 Tab Navigator 中重新访问特定屏幕时，或当用户在我们的应用程序中轻击时跟踪用户事件时，这对于进行其他 API 调用很有用。

我们可以采用多种方法：

使用事件侦听器侦听“focus”事件。 使用 react-navigation 提供的 useFocusEffect hook。 使用 react-navigation 提供的 useIsFocused hook。

# 使用“focus”事件侦听器触发动作

我们还可以使用事件监听器来监听“focus”事件。 设置事件监听器后，我们还必须在卸载屏幕时停止监听事件。

使用这种方法，我们只能在屏幕聚焦时调用动作。 这对于执行诸如记录屏幕视图以进行分析之类的操作很有用。

例子：

import * as React from "react";
import { View } from "react-native";

function ProfileScreen({ navigation }) {
  React.useEffect(() => {
    const unsubscribe = navigation.addListener("focus", () => {
      // The screen is focused
      // Call any action
    });

    // Return the function to unsubscribe from the event so it gets removed on unmount
    return unsubscribe;
  }, [navigation]);

  return <View />;
}

在大多数情况下，建议使用 useFocusEffect hook，而不是手动添加侦听器。 有关详情，请参见下文。

# 使用 useFocusEffect hook 触发动作

React Navigation 提供了一个 hook，可以在屏幕聚焦时运行效果，并在屏幕聚焦时清理效果。 这对于以下情况很有用：添加事件侦听器，在屏幕变为焦点时通过 API 调用获取数据或在屏幕进入视图后需要执行的任何其他操作。

当我们试图在页面未聚焦时停止某些操作（例如停止播放视频或音频文件或停止跟踪用户的位置）时，这特别方便。

import { useFocusEffect } from "@react-navigation/native";

function Profile({ userId }) {
  const [user, setUser] = React.useState(null);

  useFocusEffect(
    React.useCallback(() => {
      const unsubscribe = API.subscribe(userId, (user) => setUser(data));

      return () => unsubscribe();
    }, [userId])
  );

  return <ProfileContent user={user} />;
}

# 使用 useIsFocused hook 重新渲染屏幕

React Navigation 提供了一个 hook，该 hook 返回一个布尔值，该布尔值指示屏幕是否聚焦。

当屏幕聚焦时，该 hook 将返回 true；当组件不再聚焦时，该 hook 将返回 false。 这使我们能够根据用户是否在屏幕上来有条件地渲染某些内容。

当我们聚焦和聚焦屏幕时，useIsFocused hook 将导致我们的组件重新渲染。 使用此 hook 组件可能会在屏幕进入焦点和焦点未对准时引入不必要的组件重新渲染。 这可能会导致问题，具体取决于我们要求重点关注的操作类型。 因此，我们建议仅在需要触发重新渲染时才使用此 hook 。 对于诸如订阅事件或获取数据的副作用，请使用上述方法。

import * as React from "react";
import { Text } from "react-native";
import { useIsFocused } from "@react-navigation/native";

function Profile() {
  // This hook returns `true` if the screen is focused, `false` otherwise
  const isFocused = useIsFocused();

  return <Text>{isFocused ? "focused" : "unfocused"}</Text>;
}
