# 主题

主题允许你改变 React Navigation 提供的各种组件的颜色。你可以使用主题:

  • 定制颜色匹配您的品牌
  • 根据一天中特定的时间或用户的偏好提供明暗主题

# 基本用法

要传递自定义主题，可以将 theme 传递给导航容器。

import * as React from "react";
import { NavigationContainer, DefaultTheme } from "@react-navigation/native";

const MyTheme = {
  ...DefaultTheme,
  colors: {
    ...DefaultTheme.colors,
    primary: "rgb(255, 45, 85)",
  },
};

export default function App() {
  return (
    <NavigationContainer theme={MyTheme}>{/* content */}</NavigationContainer>
  );
}

您可以动态更改 theme，所有组件将自动更新以反映新主题。如果没有提供 theme，将使用默认主题。

一个主题是一个 JS 对象，它包含一个要使用的颜色列表。它包含以下属性:

  • dark (boolean): 无论这是黑暗主题还是光明主题
  • colors (object): react navigation 组件使用的各种颜色:
    • primary (string): 应用程序用于为各种元素着色的原色。通常你会想要使用你的品牌颜色。
    • background (string):各种背景的颜色，如屏幕的背景色。
    • card (string): 类卡片元素的背景颜色，如页眉、标签栏等。
    • text (string): 各种元素的文本颜色。
    • border (string): 边框的颜色，例如页眉边框、标签栏边框等。
    • notification (string): 选项卡导航角标的颜色。

在创建自定义主题时，您将需要提供所有这些属性。

主题实例

const MyTheme = {
  dark: false,
  colors: {
    primary: "rgb(255, 45, 85)",
    background: "rgb(242, 242, 242)",
    card: "rgb(255, 255, 255)",
    text: "rgb(28, 28, 30)",
    border: "rgb(199, 199, 204)",
    notification: "rgb(255, 69, 58)",
  },
};

提供一个主题将处理所有官方导航器的样式。React Navigation 还提供了一些工具来帮助您定制这些导航器，导航器中的屏幕也可以使用主题。

# 内置主题

随着操作系统增加了对明暗模式的内置支持，支持暗模式并不是为了跟上潮流，而是为了符合用户对应用运行方式的一般期望。为了提供对明暗模式的支持，这与操作系统的默认设置是一致的，这些主题被内置在 React 导航中。

你可以像这样导入默认和暗主题:

import { DefaultTheme, DarkTheme } from "@react-navigation/native";

# 使用操作系统首选项

react-native 目前支持在 0.63 中引入的内核(通过外观 API (opens new window))中检测操作系统颜色方案首选项。如果您使用的版本不支持它，您可以使用 react-native-appearance (opens new window)。

你需要 ios13 才能通过系统设置切换到暗模式。需要注意的是，Android 11 也支持它，但一些自定义层不能轻松切换。

注意:如果你使用 Expo 管理工作流，这需要 SDK 35+

import { useColorScheme } from "react-native";
import {
  NavigationContainer,
  DefaultTheme,
  DarkTheme,
} from "@react-navigation/native";

export default () => {
  const scheme = useColorScheme();

  return (
    <NavigationContainer theme={scheme === "dark" ? DarkTheme : DefaultTheme}>
      {/* content */}
    </NavigationContainer>
  );
};

# 在您自己的组件中使用当前主题

要访问在导航容器:中呈现的任何组件中的主题，你可以使用 useTheme 钩子。它返回主题对象:

import * as React from "react";
import { TouchableOpacity, Text } from "react-native";
import { useTheme } from "@react-navigation/native";

// Black background and white text in light theme, inverted on dark theme
function MyButton() {
  const { colors } = useTheme();

  return (
    <TouchableOpacity style={{ backgroundColor: colors.card }}>
      <Text style={{ color: colors.text }}>Button!</Text>
    </TouchableOpacity>
  );
}
