# TabActions

TabActions 是一个对象，其中包含用于生成特定于基于选项卡的导航器的操作的方法。 其方法扩展了 NavigationActions 中可用的操作。

# jumpTo

jumpTo 动作可用于跳至选项卡导航器中的现有路线。

  • name - string - 跳转到的路由名称。
  • params - object - 合并到目标路由的屏幕参数（通过 route.params 在推送的屏幕中找到）。
import { TabActions } from "@react-navigation/native";

const jumpToAction = TabActions.jumpTo("Profile", { user: "Satya" });

navigation.dispatch(jumpToAction);
