# useLinkBuilder

注意：此 API 是试验性的，可能会在次要版本中进行更改。

useLinkBuilder 钩子让我们构建一个路径，以用于当前导航器状态下的屏幕链接。 它返回一个函数，该函数接受 name 和 params 以使屏幕聚焦，并根据linking选项返回路径。

import { Link, CommonActions, useLinkBuilder } from "@react-navigation/native";

// ...

function DrawerContent({ state, descriptors }) {
  const buildLink = useLinkBuilder();

  return state.routes((route) => (
    <Link
      to={buildLink(route.name, route.params)}
      action={CommonActions.navigate(route.name)}
    >
      {descriptors[route.key].options.title}
    </Link>
  ));
}

该 hook 旨在用于导航器中以显示指向其中各个页面的链接，例如抽屉式导航器和选项卡式导航器。 如果要构建自定义导航器，自定义抽屉内容，自定义标签栏等，则可能要使用此钩子。

有几件重要的事情要注意：

  • 目标屏幕必须出现在当前导航器中。 它不能在父导航器中或嵌套在子导航器中。
  • 它仅用于自定义导航器中，以确保它们可在多个应用程序中重复使用。 对于常规应用程序代码，请直接使用路径，而不是为屏幕构建路径。
