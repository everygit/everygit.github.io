# useIsFocused

我们可能想根据屏幕的当前焦点状态渲染不同的内容。 该库将导出一个 useIsFocused 挂钩，以使此操作更容易：

import { useIsFocused } from "@react-navigation/native";

// ...

function Profile() {
  const isFocused = useIsFocused();

  return <Text>{isFocused ? "focused" : "unfocused"}</Text>;
}

请注意，使用此挂钩可以在更改焦点时触发屏幕的重新渲染。 如果屏幕很沉重，这可能会在动画期间导致延迟。 您可能想要提取昂贵的零件来分离组件，并使用 React.memo 或 React.PureComponent 最小化它们的重新渲染。

# 与类组件一起使用

您可以将类组件包装在功能组件中以使用钩子：

class Profile extends React.Component {
  render() {
    // Get it from props
    const { isFocused } = this.props;
  }
}

// Wrap and export
export default function (props) {
  const isFocused = useIsFocused();

  return <Profile {...props} isFocused={isFocused} />;
}
