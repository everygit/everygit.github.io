# Navigation events

您可以侦听 React Navigation 发出的各种事件以获取某些事件的通知，并且在某些情况下，请覆盖默认操作。很少有适用于每个导航器的核心事件，以及仅适用于某些导航器的特定于导航器的事件。

以下是每个导航器可用的内置事件：

  • focus - 屏幕聚焦时发出此事件
  • blur - 屏幕失焦时会发出此事件
  • beforeRemove（仅适用于 5.7 及更高版本） - 当用户离开屏幕时会发出此事件，有机会阻止用户离开屏幕
  • state（高级） - 导航器的状态更改时，将发出此事件

除此之外，每个导航器都可以发出自己的自定义事件。例如，堆栈导航器发出 transitionStart 和 transitionEnd 事件，选项卡导航器发出 tabPress 事件等。您可以在各个导航器的文档中找到有关所派发事件的详细信息。

注册为事件侦听器的每个回调都会接收一个事件对象作为其参数。事件对象包含几个属性：

data - 有关导航器传递的事件的其他数据。如果未传递任何数据，则可以是 undefined。 target - 应该接收事件的屏幕的路由。对于某些事件，如果该事件与特定屏幕无关，则可能 undefined。 preventDefault - 对于某些事件，事件对象上可能有一个 preventDefault 方法。调用此方法将阻止事件执行默认操作（例如切换 tabPress 上的选项卡）。仅在某些事件（例如 tabPress）中支持防止操作，并且不适用于所有事件。

要记住的一件事是，您只能收听直接父导航器中的事件。例如，如果您尝试在嵌套在选项卡中的堆栈内部的屏幕中添加侦听器，则不会获得 tabPress 事件。如果您需要侦听父级导航器的事件，则可以使用 navigation.dangerouslyGetParent() 获取对父级导航器的 navigation prop 的引用，并添加一个侦听器。

有两种监听事件的方法：

# navigation.addListener

在屏幕内部，您可以使用 addListener 方法在 navigation prop 上添加侦听器。 addListener 方法带有两个参数：事件的类型和在事件上要调用的回调。 它返回一个可以调用以取消订阅事件的函数。

例子：

const unsubscribe = navigation.addListener("tabPress", (e) => {
  // Prevent default action
  e.preventDefault();
});

通常，您会在 React.useEffect 中为功能组件添加一个事件侦听器。 例如：

function Profile({ navigation }) {
  React.useEffect(() => {
    const unsubscribe = navigation.addListener("focus", () => {
      // do something
    });

    return unsubscribe;
  }, [navigation]);

  return <ProfileContent />;
}

取消订阅功能可以作为效果中的清除功能返回。

对于类组件，您可以在 componentDidMount 生命周期方法中添加事件，并在 componentWillUnmount 中退订：

class Profile extends React.Component {
  componentDidMount() {
    this._unsubscribe = navigation.addListener("focus", () => {
      // do something
    });
  }

  componentWillUnmount() {
    this._unsubscribe();
  }

  render() {
    // Content of the component
  }
}

# listeners prop on Screen

有时，您可能想从定义导航器的组件中而不是在屏幕内部添加侦听器。 您可以使用 Screen 组件上的 listeners 属性来添加侦听器。 侦听器属性将事件名称作为键的对象以及侦听器的回调作为值的对象。

例子：

<Tab.Screen
  name="Chat"
  component={Chat}
  listeners={{
    tabPress: (e) => {
      // Prevent default action
      e.preventDefault();
    },
  }}
/>

您还可以传递一个回调，该回调返回带有侦听器的对象。 它将接收导航和路由作为参数。

例子：

<Tab.Screen
  name="Chat"
  component={Chat}
  listeners={({ navigation, route }) => ({
    tabPress: (e) => {
      // Prevent default action
      e.preventDefault();

      // Do something with the `navigation` object
      navigation.navigate("AnotherPlace");
    },
  })}
/>
