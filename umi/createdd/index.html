# 创建项目

使用官方工具创建项目

# 脚手架

npx @umijs/create-umi-app

# 安装依赖

yarn

# 启动项目

yarn start

# 全局安装

npm install -g @umijs/create-umi-app

# 调用本地脚手架

create-umi-app
