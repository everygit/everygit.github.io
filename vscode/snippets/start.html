# 开始

位置

文件 -> 首选项 -> 用户片段

创建一个新的用户片段

加一个

	"fc": {
		"prefix": "fc",
		"body":[
			"import React from \"react\";",
			"",
			"const $1 = () => {",
			"\treturn <div>$2</div>",
			"}",
			"",
			"export default $1;"
		]
	},

创建完成后，会有提示

光标会停留在 $1 的位置，然后，可是输入组件名称，按 tab 键，然后再进入 $2 进行组件编辑
