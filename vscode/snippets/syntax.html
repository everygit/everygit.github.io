# 语法

在 "body" 中我们可以使用特殊的结构来控制游标和插入的文字

# 制表符

使用制表符，可以使编辑器光标在代码段内移动。使用$1，$2 指定游标位置。这个数字是访问制表符的顺序，而$0 表示最后的游标位置。同一制表符的多次出现被同步链接和更新。

# 占位符

占位符是带有值的制表符，占位符的文本可以被插入或者选择，并且可以使用嵌套

# 选择

占位符的值可以选择，如 ${3|get,post,put,delete|}

# 变量变换

在插入变量之前修改变量的值。转换的定义包括三个部分:

  1. 与变量值匹配的正则表达式，或无法解析变量时匹配的空字符串。
  2. 允许从正则表达式引用匹配组的“格式字符串”。格式字符串允许有条件的插入和简单的修改。
  3. 传递给正则表达式的选项。e.g. ig 忽略大小写和全局模式
${TM_FILENAME/(.*)\\..+$/$1/}
  |           |         |  |
  |           |         |  |-> no options
  |           |         |
  |           |         |-> references the contents of the first
  |           |             capture group
  |           |
  |           |-> regex to capture everything before
  |               the final `.suffix`
  |
  |-> resolves to the filename

如果当前的文件名为 config.js，那么根据变量获取的结果就是 config

注意： \ 需要使用 \\ 来代替。

# 占位符变换

与变量的变换一样，占位符也可以变换值，以 useState 为例：

{
  "useState": {
    "key": "useState",
    "prefix": "useStateSnippet",
    "body": [
      "const [${1:first}, set${1/(.*)/${1:/capitalize}/}] = useState(${2:second})"
    ],
    "scope": "typescript,typescriptreact,javascript,javascriptreact"
  }
}

# 变换示例

假设 example-123.456-TEST.js

栗子 输出 说明
"${TM_FILENAME/[\.]/_/}" example-123_456-TEST.js Replace the first . with _
"${TM_FILENAME/[\.-]/_/g}" example_123_456_TEST_js Replace each . or - with _
"${TM_FILENAME/(.*)/${1:/upcase}/}" EXAMPLE-123.456-TEST.JS Change to all uppercase
"${TM_FILENAME/[^0-9^a-z]//gi}" example123456TESTjs Remove non-alphanumeric characters

# 语法小结

any         ::= tabstop | placeholder | choice | variable | text
tabstop     ::= '$' int
                | '${' int '}'
                | '${' int  transform '}'
placeholder ::= '${' int ':' any '}'
choice      ::= '${' int '|' text (',' text)* '|}'
variable    ::= '$' var | '${' var '}'
                | '${' var ':' any '}'
                | '${' var transform '}'
transform   ::= '/' regex '/' (format | text)+ '/' options
format      ::= '$' int | '${' int '}'
                | '${' int ':' '/upcase' | '/downcase' | '/capitalize' | '/camelcase' | '/pascalcase' '}'
                | '${' int ':+' if '}'
                | '${' int ':?' if ':' else '}'
                | '${' int ':-' else '}' | '${' int ':' else '}'
regex       ::= JavaScript Regular Expression value (ctor-string)
options     ::= JavaScript Regular Expression option (ctor-options)
var         ::= [_a-zA-Z] [_a-zA-Z0-9]*
int         ::= [0-9]+
text        ::= .*
