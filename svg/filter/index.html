# 概述

滤镜通过 <filter> 元素进行定义，并且置于 <defs> 区块中。

# 实例

一个矩形的阴影效果

<svg width="1000" height="500" style="background-color:#eeeeee">
  <defs>
    <filter
      id="ff"
      filterUnits="userSpaceOnUse"
      x="100"
      y="100"
      width="250"
      height="200"
    >
      <feGaussianBlur in="SourceAlpha" stdDeviation="4" result="blur" />
      <feOffset in="blur" dx="4" dy="4" result="offsetBlur" />
    </filter>
  </defs>
  <rect x="100" y="100" width="200" height="100" fill="red" filter="url(#ff)" />
  <rect x="100" y="100" width="200" height="100" fill="red" />
</svg>

对于滤镜的元素，有几个基本的属性

  • in 输入源 SourceAlpha 是指原图像的 alpha 通道，SourceGraphic 指原图像，也可以使用其他元素的 result 值
  • result 临时缓冲区的命名

有的元素可能会用更多的输入源，例如 in2

# in 属性的值

值 描述
SourceGraphic 图像本身作为元素的输入源
SourceAlpha 图像的透明度作为元素的输入源
BackgroundImage 当前 filter 元素底下区域的快照作为输入源
BackgroundAlpha 同上，使用期元素的透明度作为输入源
