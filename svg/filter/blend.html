# feBlend

把两个对象组合在一起，使它们受特定的混合模式控制。这类似于图像编辑软件中混合两个图层

# mode

混合模式

# normal

最后的颜色是顶部的颜色，不管底部是什么颜色。这种效果就像两张不透明的纸重叠在一起。

# multiply（正片叠加）

最后的颜色是上下颜色相乘的结果。一个黑色的图层导致最后一个黑色的图层，一个白色的图层导致没有变化。其效果就像印在透明胶片上的两幅图像重叠。

公式：混合色 × 基色 / 255=结果色

# screen（滤色模式）

最后的颜色是将颜色反转，相乘，然后反转值的结果。一个黑色的图层导致没有变化，一个白色的图层导致最后一个白色的图层。这种效果就像两幅图像照射在投影屏幕上。

# overlay（叠加模式）

如果底色较深，最后的颜色是乘法的结果，如果底色较浅，则是 screen 的结果。这种混合模式相当于 hard-light，但是层被交换了

# darken（变暗模式）

最后的颜色由每个颜色通道中最暗的值组成。

# lighten（变亮模式）

最后的颜色由每个颜色通道中最亮的值组成。

# color-dodge（颜色减淡模式）

最后的颜色是底色除以底色的倒数的结果。一个黑色的前景导致没有变化。与背景色相反的前景会导致完全明亮的颜色。这种混合模式类似于 screen，但前景只需要像背景的反方向一样亮，就可以创建一个完全明亮的颜色。

# color-burn

最终的颜色是将底部颜色反转，将值除以顶部颜色，并反转该值的结果。白色的前景导致没有变化。前景与背景的反色导致一个黑色的最终图像。这种混合模式类似于 multiply，但前景只需要像背景的反色一样暗，使最终图像黑色。

# hard-light

如果顶部颜色较深，则最终颜色为 multiply 的结果;如果顶部颜色较浅，则颜色为 screen 的结果。这个混合模式相当于 overlay，只是层被交换了。这种效果类似于在背景上用强光照射。

# soft-light

最后的颜色类似于 hard-light，但更柔和。这种混合模式的行为类似于 hard-light 模式。这种效果类似于在背景上使用扩散的聚光灯

# difference

最后的颜色是两种颜色中较暗的那一种减去较浅的那一种。一个黑色的图层没有效果，而一个白色的图层反转另一个图层的颜色。

# exclusion

最后的颜色与 difference 相似，但对比度较低。使用 difference 时，一个黑色的图层没有效果，而一个白色的图层反转另一个图层的颜色。

# hue（色相模式）

最后的颜色有顶部颜色的色调，同时使用底部颜色的饱和度和亮度。

# saturation（饱和度模式）

最后的颜色具有顶部颜色的饱和度，同时使用底部颜色的 色相 和 饱和度。纯灰色背景，没有饱和度，将没有效果。

# color（颜色模式）

最终的颜色具有顶部颜色的色相和饱和度，同时使用底部颜色的亮度。这种效果保留了灰度级别，可以用来给前景着色。

# luminosity（亮度模式）

最后的颜色具有顶部颜色的亮度，同时使用底部颜色的色相和饱和度。这个混合模式相当于 color，但是层被交换了。

# 扩展

#div {

  height: 300px;
  background: url("br.png"), url("tr.png");
  background-blend-mode: normal;
}
