# 图案

类似于 html 中的背景图片的平铺，跟渐变一样，需要放在 <defs> 标签中。

# 示例

<svg width="200" height="200">
  <defs>
    <pattern id="Pattern" x="0" y="0" width=".25" height=".25">
      <rect x="10" y="10" width="20" height="20" fill="skyblue" />
    </pattern>
  </defs>
  <rect
    fill="url(#Pattern)"
    stroke="black"
    x="0"
    y="0"
    width="200"
    height="200"
  />
</svg>
  • pattern 定义图案
  • x y 图案的位置，这是一个相对的位置，单位受 patternUnits 影响
  • width height 图案的宽度和高度，单位受 patternUnits 影响
  • patternUnits 属性单元，决定了 x y width height 的单位
    • objectBoundingBox 默认值 0 0 1 1 相对于整个盒子，上面的 0.25 就是一个要填充的容器的 1/4
    • userSpaceOnUse 绝对单元，与 svg 同一坐标系
  • patternContentUnits 内容的的属性单元
    • userSpaceOnUse 默认值， 绝对单元，与 svg 同一坐标系
    • objectBoundingBox 默认值 0 0 1 1 相对于整个盒子，要避免使用
