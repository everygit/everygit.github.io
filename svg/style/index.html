# 填充 fill

填充类似于 background-color，在 svg 的 css 定义中，也要使用 fill 来代替 background-color

<rect x="10" y="10" width="100" height="100" fill="purple" fill-opacity="0.5" />
  • fill 填充的颜色，可以是颜色名，如 red rgb 值，如 rgb(255, 0, 0)或者 16 进制、rgba 等
  • fill-opacity 填充的透明度，值为 0-1
  • fill-rule 用来确定一个多边形内部区域的算法( 实现空心效果就靠这个东东了 )
    • nonzero 当前点向外发射一条射线，相交的线顺时针 +1，逆时针 -1，最后总数非 0，则表示内部，反之为外部
    • evenodd 当前点向外发射一条射线，相交 + 1，总数为奇数时表示内部，反之为外部

# 边框 stroke

类似于 border

<rect
  x="10"
  y="10"
  width="100"
  height="100"
  stroke="blue"
  stroke-opacity="0.8"
/>
  • stroke 边框的颜色，同 fill
  • stroke-width 边框的宽度
  • stroke-opacity 边框的透明度
  • stroke-linecap 边框终点的样式
    • butt 直边结束线段，终点的样式不会影响整个线段的长度
    • square 终点是一个正方形，长度会比线段长出一个宽度（两个半个的宽度）
    • round 终点是一个半圆，也会比原来的长度长一个宽度
  • stroke-linejoin 折线的连接样式
    • miter 默认样式，连接处形成一个尖角
    • round 圆角连接，实现了平滑处理
    • bevel 倒角，类似于拐弯抹角的意思，具体看下图吧
  • stroke-dasharray 虚线边框的设置
  • stroke-dashoffset 指定了 dash 模式到路径开始的距离
  • stroke-miterlimit 尖角的一个极限值，超出后，会表现为 bevel （比较好理解）
