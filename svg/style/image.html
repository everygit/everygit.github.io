# foreignObject

包含来自不同的 XML 命名空间的元素。在浏览器的上下文中，很可能是 XHTML / HTML，简单点来说，就是可以在 svg 中插入 html 代码

<svg height="300">
  <circle cx="100" cy="100" r="100" fill="red" />
  <foreignObject x="50" y="100" width="100" height="50">
    <div style="text-align:center;">
      <a href="https://www.baidu.com">百度</a>
    </div>
  </foreignObject>
</svg>
  • x y 位置， 默认 0
  • width height 大小，默认 auto
