# 变形

跟 css 中的 transform 一样

# 平移

向右移动 30，向下移动 40

<rect x="0" y="0" width="10" height="10" transform="translate(30,40)" />

# 旋转

旋转 45 度

<rect x="20" y="20" width="20" height="20" transform="rotate(45)" />

单位为度

如果几个变形要同时用是这样的 transform="translate(100, 100),rotate(30),scale(2)"

# 斜切

<rect x="0" y="0" width="200" height="200" transform="skewX(20),skewY(20)" />

skewX 和 skewY 分别代表两个方向的斜切

# 缩放

<rect x="0" y="0" width="200" height="200" transform="scale(0.5)" />

以上基本上和 css 都差不多
