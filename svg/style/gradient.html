# 渐变

为了满足渐变小王子的要求，贫道准备研究一下渐变。渐变可以应用于边框与填充，在 <defs> 中定义；渐变分为线性渐变和径向渐变，下面就是见证奇迹的时刻。

# 线性渐变

所谓的线性渐变，就是沿着一条直线改变颜色或透明度

# 实例

<svg width="500" height="500">
  <defs>
    <linearGradient id="gradient" x1="0" x2="1" y1="0" y2="0">
      <stop offset="0%" stop-color="#ff0000" />
      <stop offset="100%" stop-color="#00ffff" />
    </linearGradient>
  </defs>
  <rect x="50" y="50" width="400" height="400" fill="url(#gradient)" />
</svg>
  • linearGradient 指定一个线性渐变
    • id 线性渐变的 id，在填充或者描边时会被引用
    • x1 x2 y1 y2 表达渐变的方向和起始位置
  • stop 颜色中值
    • offset 渐变的偏移量
    • stop-color 颜色中值
    • stop-opacity 透明度

https://codepen.io/everygit/pen/MWoLVQO (opens new window)

表达方式会有很多种，诸如 class 、xlink 等，这里不做介绍了，想了解到官网

# 径向渐变

一个点向四周发散

# 实例

<svg width="700" height="500">
  <defs>
    <radialGradient id="radial">
      <stop offset="0%" stop-color="red" />
      <stop offset="100%" stop-color="green" stop-opacity="0" />
    </radialGradient>
  </defs>
  <rect x="300" y="50" width="200" height="200" fill="url(#radial)" />
</svg>
  • radialGradient 定义一个径向渐变
    • cx cy r 圆心的 x、y 点及圆的半径
    • fx fy 焦点的位置，焦点必须在圆内，否则不能正确展示
  • stop 颜色中值
    • offset 偏移量
    • stop-color 颜色中值
    • stop-opacity 透明度

https://codepen.io/everygit/pen/MWoLVQO (opens new window)

# spreadMethod

该属性控制了当渐变到达终点的行为

# pad

默认效果，最终的偏移颜色填充对象剩下的空间

# repeat

在剩余的空间里，重复渐变

# reflect

在剩余的空间里，继续填充，方向与前面的渐变方向相反，从 100% 到 0%

# userSpaceOnUse

# objectBoundingBox
