# 曲线

曲线分为大曲和特曲，啊呸，曲线分为贝塞尔曲线和弧形；贝塞尔曲线的话，用过 Photoshop 的话，里面的钢笔用的就是贝塞尔曲线了；这里不做太多的解释，后面找个时间具体研究下贝塞尔曲线，以及其中的一些数学公式。

# 三次贝塞尔

三次贝塞尔曲线需要定义一个点和两个控制点，三次贝塞尔使用 C 命令

# 代码

C x1 y1, x2 y2, x y
c dx1 dy1, dx2 dy2, dx dy
  • x1 y1 第一个控制点的位置坐标
  • x2 y2 第二个控制点的位置坐标
  • x y 终点的位置坐标

简单点理解就好了

# S 命令

S x2 y2, x y
s dx2 dy2, dx dy

当 S 命令在 C 命令或者 S 命令后面，那么它的第一个控制点会是前一个曲线的第二个控制点的对称点；当 S 是孤独的存在时，当前起始点就是第一个控制点

# 实例

https://codepen.io/everygit/pen/MWozLgy (opens new window)

# 二次贝塞尔

二次贝塞尔只有一个控制点

# 语法

Q x1 y1, x y
q dx1 dy1, dx dy
  • x1 y1 控制点位置坐标
  • x y 终点位置坐标

# T 命令

T x y
t dx dy

和 S 命令的效果类似，控制点的位置也是对称的。什么？ T 命令单独出现会怎样，不会怎么，就是一条直线而已

# 弧形

弧形是圆或者椭圆的一部分，画弧形会用到 A 命令

# 代码

A rx ry x-axis-rotation large-arc-flag sweep-flag x y
a rx ry x-axis-rotation large-arc-flag sweep-flag dx dy
  • rx 长轴半径
  • ry 短轴半径
  • x-axis-rotation 旋转的角度
  • large-arc-flag 是否为大角圆弧 0 表示小角圆弧 1 表示大角圆弧
  • sweep-flag 弧线的防线 0 表示从起点到终点为逆时针，1 表示从起点到终点为顺时针
  • x y 终点的位置坐标
