# 直线命令

绘制直线的命令， L、H、V、Z

# 语法

分别三种命令

# Line to

L x y
l dx dy

从当前画笔的位置，划线到 (x, y) 坐标

# H 命令

L 的简写，表示绘制水平线，

H x
h dx

因为只关注 x 方向，所以不需要 y 的位置

# V 命令

L 的简写，表示绘制垂直线

V y
v dy

因为值关注 y 方向，所以不需要 x 的位置

# Z 命令

从当前的画笔位置绘制直线到起点位置

Z
z

这个玩意其实不区分大小写，怎么好看怎么写就可以了
