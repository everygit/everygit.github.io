# 概述

# 主要组成

# Displayable

继承自 Element ，可见元素，在 Element 之上增加了 style ，用于控制元素的样式。

# Group

组。Group 是一个容器，可以插入子节点，Group 的变换也会被应用到子节点上。

# Element

元素，可以被添加到场景中。可以进行变换（Transformable）、支持动画（Animatable），支持事件（Eventful）

# 扩展

# Path

路径，绘制所有图形的基类，继承自 Displayable
