# justify-content

用于设置整个内容区域在容器里面的水平位置

内容的总宽度小于容器的宽度是生效

# 语法

假设文档流的方向为水平方向

justify-content: stretch | start | end | center | space-between | space-around |
  space-evenly;

太棒了，来自于星星的你，一眼就看出来跟 flex 一毛一样

  • stretch 拉伸，宽度填满 grid 容器
  • start 左对齐（文档流方向为水平方向，下同）
  • end 右对齐
  • center 居中对齐
  • space-between 两端对齐，且平均分配间隔
  • space-around 每个子项的左右间隔相同， 子项之间的距离是两侧子项与容器间隔的两倍
  • space-evenly 每个子项之间的间隔是相同的，包括两侧与容器边缘的间隔
