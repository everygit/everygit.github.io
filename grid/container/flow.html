# grid-auto-flow

控制没有明确指定位置的 grid 子项的放置方式

# 语法

.container {
  grid-auto-flow: row | column | row dense | column dense;
}
  • row 默认值，没有指定位置的子项以水平方向排列优先。
  • column 没有指定位置的子项以垂直方向排列优先。
  • dense 自动排列启用“密集”打包算法，根据 row 和 column 分别以"先行后列"和"先列后行"的形式尽量填满空格

dense 会导致顺序发生变化，如果对顺序有严格要求的话，要尽量避免使用
