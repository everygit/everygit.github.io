# grid-area

指定子项放在哪个区域里，配合 grid-template-areas 使用

# 语法

.item {
  grid-area: <name> | <row-start> / <column-start> / <row-end> / <column-end>;
}
  • <name> 区域名称。由 grid-template-areas 属性创建。
  • <row-start> / <column-start> / <row-end> / <column-end>占据网格区域的纵横起始位置。

# 实例

关于 name 的使用方法可以参考 grid-template-areas

第二种使用方法

.item-1 {
  grid-area: 1 / 1 / 3 / 3;
}

合并了第一行第二行，第一列第二列，相当于四个单元格合并成为一个。
