# grid-column-start, grid-column-end, grid-row-start 和 grid-row-end

表示 grid 子项所占据的区域的起始和终止位置，包括水平方向和垂直方向。

# 语法

.item {
  grid-column-start: <number> | <name> | span <number> | span <name> | auto;
  grid-column-end: <number> | <name> | span <number> | span <name> | auto;
  grid-row-start: <number> | <name> | span <number> | span <name> | auto;
  grid-row-end: <number> | <name> | span <number> | span <name> | auto;
}
  • <number> 起止与第几条网格线。
  • <name> 自定义的网格线的名称。
  • span <number> 表示当前网格会自动跨越指定的网格数量。
  • span <name> 表示当前网格会自动扩展，直到命中指定的网格线名称。
  • auto 全自动，包括定位，跨度等

# 实例

跨两列， 1 表示一个网格线，就是容器的左侧边缘，3 就是第三条网格线，第二个网格的右侧边缘

.item {
  grid-column-start: 1;
  grid-column-end: 3;
}

使用网格名称，在这里同样还是跨两列

.container {
  grid-template-columns: [网格边边] 200px [lsp] 1fr [吃桃子凉凉] 2fr;
}

.item {
  grid-column-start: 网格边边;
  grid-column-end: 吃桃子凉凉;
}

使用 span，这里还是跨两列，真气人 O(∩_∩)O 哈哈~

.item {
  grid-column-start: 1;
  grid-column-end: span 2;
}
