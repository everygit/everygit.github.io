# window 安装 nvm

# 前提

卸了老版本吧。顶多就是很多东西重新装一下。

# 下载地址

https://github.com/coreybutler/nvm-windows/releases

# 选择文件

nvm-setup.zip

# 安装

随意

# 修改 settings.txt

文件位于 nvm 的安装目录中

安装时会出现两个目录，第一个是 nvm 第二个是 nodejs 的安装地址

node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

# 基本命令

# 查询已安装列表

nvm ls
nvm list

# 安装 node 版本

nvm install 10.8.0

# 选择版本

nvm use 10.8.0

# 查询可下载版本

nvm list available
